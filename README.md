# 地球视差官网

一个具有3D地球、视差效果和色散分离的现代化官网。

## 特性

- 🌍 自动旋转的3D地球
- ✨ 大小不同的发光节点
-    主色调 #FF7F50 (珊瑚色)
-    背景色 #EFEEEB (米白色)
- 🌊 鼠标悬停时的视差效果和色散分离
-    周围漂浮的水滴效果
- 📱 响应式设计

## 技术栈

- HTML5
- CSS3 (动画、渐变、滤镜)
- JavaScript (ES6+)
- Three.js (3D渲染)
- WebGL

## 使用方法

1. 下载所有文件到同一目录
2. 在浏览器中打开 `index.html`
3. 移动鼠标到地球上体验视差效果

## 浏览器兼容性

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 自定义

- 修改 `styles.css` 中的颜色变量
- 调整 `script.js` 中的地球大小和旋转速度
- 在 `index.html` 中更改文本内容
